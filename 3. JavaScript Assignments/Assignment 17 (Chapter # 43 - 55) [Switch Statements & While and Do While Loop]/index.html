Placing Script Element in HTML 5

In HTML 5 we can add the script element at any portion of the html page like header / starting of the body / ending of the body / main element of the body.

But good portion of Script element is ending of the body.

Two ways adding the Script element at ending of the body.
  • Create the Script element at ending of the body.
  • Create script element at any portion of the html page but using Defer Attribute

    This Defer Attribute excute the Script at the last of html page.

Comment in HTML File

<!-- --> : Only one way for adding the comment in HTML File

Comment in CSS File

/* */ :Only one way for adding the comment in CSS File

Comment in JS File

There are two ways for adding the comment in JS File
  • // : For single line comment.
  • /* */ : For multi line comment.
